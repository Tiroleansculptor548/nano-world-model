## 🚀 Getting started

The main window shows a control panel and a viewing area. The control panel contains several buttons. Each button represents a specific experiment mode.

*   **World Prediction**: This mode allows the software to guess what happens next in a scene.
*   **Video Generation**: This mode creates streaming sequences based on current input parameters.
*   **Robot Logic**: Use this to test movement planning for simulated robot hands.
*   **Video Stream**: This activates the visual interface to watch the model at work.

Click any of these buttons to begin a simulation. The software displays progress bars when it performs complex calculations. Wait for these bars to complete before you interact with the results.

## 🔍 Understanding the features

The system utilizes several core technologies to function. These include diffusion models and prediction algorithms. While the underlying math is complex, the interface hides these details.

### Diffusion-forcing
This method guides the software toward realistic outcomes. It prevents the model from generating random or chaotic data. The software forces the output to match the patterns it learns from sample video streams.

### Planning and Control
The model can predict the best path for a robot arm within the system. You can observe how it avoids barriers. The planning feature adjusts the path in real time if you add new obstacles.

### World Model Science
The software functions as a digital representation of a physical environment. It learns how objects fall, slide, or collide. It uses this knowledge to mimic reality within the simulation window.

## 🛠 Troubleshooting common issues

If the application fails to open, verify your graphics card drivers. Visit the website of the manufacturer of your graphics card to download the latest updates. Restart your computer after you update the drivers.

If the simulation runs slowly, change the settings in the menu. Lower the resolution of the video output to reduce the demand on your computer hardware. Click the Settings icon in the top right corner to access these options. Select lower quality levels to increase the speed of the output.

If you see an error window, take a screenshot. You can open an issue on the repository website to share the error with the project maintainers. Provide as much detail as you can about what you did before the crash happened.

## 🧹 Managing disk space

The application stores cached files to improve speed. These files occupy disk space over time. If you run out of space, delete the contents of the cache folder inside the main program directory. The application creates a new cache folder automatically the next time you use it. You can safely remove the cache folder at any time.

## 💡 Usage tips

Keep your computer plugged into power while you run simulations. Laptop users should set their power mode to High Performance. This setting allows the software to use the full power of your hardware. Close other heavy programs while the simulation prepares its models. This frees up your system memory for the tasks performed by the application.

Use the zoom tool to see details in the simulation. Hover your mouse over the visual area and scroll your mouse wheel. This lets you inspect objects in the 3D space. You can view the simulation from different angles by clicking and dragging your mouse inside the main window. 

The software stops the simulation automatically when the task finishes. You can press the Stop button at any time to halt the process.
